A Reverse a Substring

思路

只要找到字符串中的连续两个字符前面的大于后面的，就说明可以，于是输出yes并交换他们俩，如果找不到，输出NO。

题目

给定一个由n个小写拉丁字母组成的字符串s。
让我们将子字符串定义为字符串的连续子段。例如，“acab”是“abacaba”的子字符串(它从位置3开始，以位置6结束)，但是“aa”或“d”不是这个字符串的子字符串。所以从位置l到位置r的字符串s的子字符串是s[l;r]=slsl+1…sr。
您必须精确地选择给定字符串的一个子字符串并将其反转(即。使s[l;r]=srsr - 1…注意，没有必要获取最小可能的字符串。
如果无法反转给定字符串的某些子字符串以获得更小的字符串，则打印“NO”。否则打印“YES”和任何合适的子字符串。
字符串x在词典上小于字符串y，如果x是y的前缀(且x≠y)，或者存在i(1≤i≤min(|x|，|y|))，则xi<yi，对于任意j(1≤j) xj=yj。这里|a|表示字符串a的长度。字符串的字典比较是由现代编程语言中的操作符<实现的。
输入：
输入的第一行包含一个整数n (2≤n≤3⋅10^5) - s的长度。
输入的第二行包含长度为n的字符串s，仅由小写拉丁字母组成。
输出：
如果无法对给定字符串的某些子字符串进行反转以获得词法上更少的字符串，则打印“NO”。否则打印“YES”和两个索引l和r(1≤l<r≤n)，表示必须反转的子字符串。如果有多个答案，可以打印任何一个。
样例：
7
abacaba
答案：
YES
2 5

B Game with Telephone Numbers

思路

最后10位是什么没关系，就看这串序列前面的位数中有几个8，如果超过（n-11）/2，则Vasya获胜。

题目

电话号码是由11位数字组成的序列，其第一个数字是8。
瓦斯亚和彼佳在玩游戏。最初它们有一个长度为n(n是奇数)的字符串，由数字组成。瓦斯亚先走一步，然后玩家轮流。玩家必须选择一个字符并将其从当前字符串中删除。例如，如果当前的字符串1121，在玩家移动之后它可能是112、111或121。当字符串长度s变为11时，游戏结束。如果得到的字符串是电话号码，则Vasya获胜，否则Petya获胜。
你必须确定Vasya是否有获胜的策略(也就是说，无论Petya在移动过程中选择了哪个角色，Vasya是否能够赢得游戏)。
输入：
第一行包含一个整数n(13≤n<105,n为奇数)——字符串的长度s。
第二行包含字符串s(|s|=n)，它只由十进制数字组成。
输出：
If Vasya has a strategy that guarantees him victory, print YES.
Otherwise print NO.
样例：
13
8380011223344
答案：
YES

C Alarm Clocks Everywhere

思路

一直理解错题解了，结果最后没做出来，难过QAQ本来很简单的，做出来肯定上分了
对x数组求差，之后对所有差求GCD，再暴力查找p数组有没有GCD%p[i]==0。若有：输出YES，y就等于x[0]，j就是i+1;若没有输出NO。

题目

伊凡现在要睡觉了，他想设定闹钟。明天有许多必要的活动，第i项将在第11分钟开始。伊凡不想跳过任何事件,所以他必须把他的闹钟,这样环在分钟x1, x2,…, xn,所以他将在每一个分钟清醒(注意,这并不重要,如果他的闹钟将环在其他任何分钟)。
伊万可以选择两个属性闹钟-第一分钟将环(表示这是y)和连续两个信号之间的时间间隔(p)表示吧。时钟被设置后,将环在分钟y, y + p + 2 p y + 3 p等等。
Ivan可以选择任意分钟作为第一个分钟，但是他不能选择任意的p值。他必须在给定的值p1,p2，…，pm中选择它(他的手机不支持任何其他设置选项)。
所以伊万选择第一分钟y当闹钟应该开始响了,连续两个信号之间的时间间隔pj以这样一种方式,它将环在给定分钟x1, x2,…, xn(和它并不重要,如果他的闹钟将环在其他任何分钟)。
你的任务是告诉第一个分钟y和索引j，这样如果Ivan用属性y和pj设置他的闹钟，它会在所有给定的分钟x1、x2、…、xn内响起，或者说不可能选择给定属性的这些值。如果有多个答案，可以打印任何一个。
输入：
入的第一行包含两个整数n和m (2≤n≤3⋅10^5, 1≤≤3⋅10^5) -事件的数量和可能的设置数量,信号之间的时间间隔。
输入的第二行包含n个整数x1,x2，…，xn(1≤xi≤10^18)，其中xi为第i个事件开始的分钟。它保证所有xi都是按递增的顺序给出的。条件x1<x2<⋯⋯<xn成立)。
输入的第三行包含m个整数p1,p2，…，pm(1≤pj≤10^18)，其中pj为两个连续信号间隔的第j个选项。
输出：
如果不可能选择这样的值y和j，因此满足所有约束，那么在第一行中打印“NO”。
否则在第一行打印“YES”。然后打印两个整数y(1≤y≤10^18)和j (1≤j≤m)在第二行,y是伊凡第一分钟的闹钟应该开始振铃和j的指数选择连续两个信号之间的时间间隔(选项编号从1米他们给出的顺序输入)。这些值的选择应该使闹钟在给定的x1、x2、……、xn分钟内都能响起。如果有多个答案，可以打印任何一个。
样例：
3 5
3 12 18
2 6 5 3 3
答案：
YES
3 4
